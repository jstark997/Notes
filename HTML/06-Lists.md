# Lists

List elements are typically either ordered (numbered) or unordered. Within each type are list item elements that are designated by the list item tag.

## List Tags

- **`<ol>`**: Ordered list.
- **`<ul>`**: Unordered list.
- **`<li>`**: List item within `<ol>` or `<ul>`.

Ordered list example:

```html
<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>
```
