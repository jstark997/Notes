# TypeScript - Operators

Operators in TypeScript:

- Arithmetic: `+`, `-`, `*`, `/`, `%`
- Assignment: `=`, `+=`, `-=`, `*=`, `/=`
- Comparison: `==`, `!=`, `===`, `!==`, `>`, `<`, `>=`, `<=`
- Logical: `&&`, `||`, `!`
- Bitwise: `&`, `|`, `^`, `~`, `<<`, `>>`, `>>>`
- Conditional (ternary): `condition ? expr1 : expr2`

Example:

```typescript
let x: number = 10;
let y: number = 5;
let result: number = x + y;
```
